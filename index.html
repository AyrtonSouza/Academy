Academy

Projeto em Spring de Sistemas Para Academia, WEB e Mobile

View project on GitHub

Descrição do Projeto

Se trata de um Sistema Voltado para academias, o foco do sistema é gerenciar os alunos da academias e os professores no que diz respeito ao treino, por isso o sistema foi divido em duas partes, a parte web para administradores da academias e administrador do site, aonde era capaz de inserir os alunos e os professores, além de inserir exercícios e aparelhos.

A Segunda Parte, é um sistema mobile, voltado para alunos e professores, aonde os professores são capazes de criar treinos para os alunos inserir as medidas dos alunos e acompanhar o rendimento por aluno, já os alunos visualizavam os treinos e inseriam somente a series e a repetição, alem de ver gráficos de rendimentos, tanto muscular quando por aparelhos.

o sistema que esta neste repositório se encontra incompleto, uma vez que não foi possível por mim possuir o sistema todo, uma vez que eu não recebi permissão do Author do projeto em mostra-lo aqui, mas esse parte do projeto foi disponibilizada para mostrar que tive participação nesse projeto

Tecnologias Abordadas

Foi utilizado nesse projeto, a linguagem java, com o spring framework(Spring MVC, Spring AOP, Spring Security) para o desenvolvimento da parte WEB, e para o desenvolvimento Mobile foi o usado o Jquery, para dinamização da pagina para qualquer dispositivo portátil
